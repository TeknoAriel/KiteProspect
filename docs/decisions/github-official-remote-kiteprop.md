# Repositorio oficial GitHub (organización kiteprop)

## Remoto canónico

```
git@github.com:kiteprop/ia-kiteprospects.git
```

Rama de trabajo: **`main`**.

## Política

- El agente y el equipo hacen **`git push origin main`** contra este remoto (no forks personales como fuente de verdad).
- CI: `.github/workflows/ci.yml` en cada push/PR a `main` / `master`.
